# jwt-template
Template for new projects using tdd for a system with user login, signup and user update using an API. Integrated with mongoDB and Docker.

## Login - SignUp - Mongo - Express -  JWT - Template


### Requirements 
1. Docker
2. Node 8.5
3. npm or yarn
4. mocha.
```
yarn add global mocha
```
5. nyc
```
yarn add global nyc
```

### To run the project
```
docker-compose up -d
```
